# Voces por la Dignidad

Blog personal de análisis y reflexión en derechos humanos, accesibilidad e inclusión social.

**Autor:** Lenin Pérez Guzmán  
**País:** Costa Rica  
**Enfoque:** Educativo, preventivo y formativo  
**Perspectiva:** No partidaria · No confrontativa · Regional

## Descripción

*Voces por la Dignidad* es un espacio personal e independiente de carácter educativo. Aquí publico reflexiones y análisis sobre derechos humanos, accesibilidad universal e inclusión social desde Costa Rica.

Este sitio pertenece y es gestionado exclusivamente por su autor. No representa a ninguna organización, partido político ni gobierno.

## Cómo desplegar en GitHub Pages

1. Crear un repositorio en GitHub (p.ej. `voces-por-la-dignidad`)
2. Subir todos los archivos a la rama `main`
3. En **Settings → Pages**, seleccionar `main` como rama fuente y `/` como directorio raíz
4. GitHub Pages publicará el sitio en `https://[usuario].github.io/voces-por-la-dignidad/`

> **Importante:** Actualizar las URLs en `sitemap.xml` con el dominio real una vez desplegado.
